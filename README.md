# Laravel Marcho Ecommerce

This is my project Laravel Marcho Ecommerce

### Used

-   [Laravel] - php framework
-   Bootstrap
-   JQuery

### Installation

```sh
$ git clone
$ composer install
// Database is : 'laravel.sql'
$ cp .env.example .env
$ php artisan key:generate
    setup DB in .env
$ npm install
$ npm run dev
$ php artisan server
```
